# SUDAN GREEN

A web application that allows users to transform their images with a green effect to show solidarity with Sudan.

## Features

- Upload images via drag-and-drop or file selection
- Apply a green effect to uploaded images
- Compare original and processed images side by side
- Download the processed green images
- Responsive design for all devices
- Arabic language support

## Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla)
- Canvas API for image processing

## How to Use

1. Visit the website
2. Upload an image by clicking on the upload area or dragging and dropping an image
3. Click the "تحويل الصورة" (Transform Image) button
4. View the original and green-processed images
5. Download the green image by clicking the "تحميل الصورة الخضراء" (Download Green Image) button

## Deployment

### Local Deployment

To run this project locally:

1. Clone the repository
2. Open the `index.html` file in your browser

### Server Deployment

To deploy to a web server:

1. Upload all files to your web server
2. Ensure the directory structure is maintained
3. No additional configuration is needed as this is a static website

### GitHub Pages Deployment

To deploy to GitHub Pages:

1. Push the repository to GitHub
2. Go to repository settings
3. Navigate to the "Pages" section
4. Select the main branch as the source
5. Save the settings and wait for the deployment to complete
